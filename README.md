# CodePilot Hub

GitHub uchun tayyor HTML, CSS va JavaScript loyihasi.

## Loyihada nimalar bor?

- Responsive landing page
- Dark / Light mode
- Project search va filter
- Button bosilganda boshqa sahifaga o‘tish
- 4 ta mini JavaScript app:
  - Todo Manager
  - Budget Calculator
  - Password Checker
  - JavaScript Quiz
- localStorage bilan ma’lumot saqlash

## Ishlatilgan texnologiyalar

- HTML5
- CSS3
- JavaScript

## GitHub'ga yuklash

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/USERNAME/CodePilot-Hub.git
git push -u origin main
```

`USERNAME` o‘rniga o‘zingizning GitHub username yozing.

## GitHub Pages qilish

1. Repository ichiga kiring
2. Settings bo‘limini oching
3. Pages bo‘limiga kiring
4. Branch: `main`, Folder: `/root` tanlang
5. Save bosing

Keyin loyiha brauzerda link orqali ochiladi.
